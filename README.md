# Crime Incident Data Analysis and Dashboard Visualization (2020–2024) Using Power BI

## Project Overview

This project focuses on analyzing crime incident data from 2020–2024 using Power BI. The dashboard identifies crime trends, victim demographics, hotspot locations, weapon usage patterns, and area-wise crime distribution.

## Tools Used

* Microsoft Power BI
* Microsoft Excel
* Power Query

## Features

* KPI Metrics
* Area-wise Crime Analysis
* Victim Gender Analysis
* Crime Type Distribution
* Weapon Usage Analysis
* Crime Hotspot Map
* Interactive Filters and Slicers

## Dashboard Visualizations

* Total Crimes KPI
* Area-wise Crime Analysis
* Victim Distribution
* Crime Type Distribution
* Weapon Analysis
* Crime Hotspot Analysis

## Dataset Timeline

2020–2024

## Outcome

The dashboard transforms raw crime data into meaningful visual insights and supports data-driven analysis and decision-making.
